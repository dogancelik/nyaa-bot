Welcome to nyaa-bot's documentation!
====================================

nyaa-bot is an IRC bot. It is written for nyaa-nyaa.com's IRC channel #nyaa-nyaa in Rizon network.

It supports:

* Channel, user, action, message specific commands
* Partial module reloading
